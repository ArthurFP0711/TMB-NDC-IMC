TMB

A Taxa metabólica basal refere-se à quantidade de energia que o corpo humano precisa para realizar funções básicas em repouso, como respiração, circulação sanguínea, manutenção da temperatura corporal e funcionamento dos órgãos vitais.

Fórmula para homens : TMB = (88.362 + (13.397 x peso) + (4.799 x altura) - (5.677 x idade))

Fórmula para Mulheres : TMB = (655 + (9.6 x peso) + (1.8 x altura) - (4.7 x idade) )

NDC

o NDC refere-se à Necessidade diária de calorias que uma pessoa precisa consumir para manter seu peso atual, levando em consideração seu nível de atividade física e outros fatores.

IMC

O Índice de Massa Corporal (IMC) é uma medida usada para determinar se uma pessoa está dentro de uma faixa de peso saudável com base na sua altura e peso.

O cálculo do IMC é feito usando a fórmula: IMC = peso(kg) / altura(m)^2
